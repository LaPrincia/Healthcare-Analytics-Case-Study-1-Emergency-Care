# 🏥 Emergency Department Operations & Patient Experience

*A Healthcare Analytics Case Study*

---

## Project Overview

This healthcare analytics case study demonstrates how operational and patient experience data can be transformed into actionable insights using Microsoft SQL Server and Tableau.

Developed for the fictional **Palmetto Regional Health System (PRHS)**, this project analyzes Emergency Department performance to support executive decision-making, improve operational efficiency, enhance patient flow, and identify opportunities to improve the patient experience.

---

## Business Challenge

Palmetto Regional Health System (PRHS) was experiencing increased Emergency Department volumes, longer patient wait times, inconsistent patient throughput, and opportunities to improve the overall patient experience.

Executive leadership required greater visibility into operational performance to identify bottlenecks, monitor key performance indicators (KPIs), and support data-driven decisions aimed at improving quality, efficiency, and patient outcomes.
